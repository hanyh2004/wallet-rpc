JSONRPC 2.0 client with CLI of crypto-currency by TypeScript for Node.js

## Supports

- BitcoinSeries(Core 0.16+)
- OmniLayer(includes Tether USDT)
- EthereumSeries(Geth 1.8.0+ || Parity 1.0.0+)
- EOS(v1.3.0 stable)
